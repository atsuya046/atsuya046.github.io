Webpackを使ってみる

Webpack

Webpack はJSモジュールの依存性管理ツール.

NodeJSのように require() 使ってmoduleをロードすることができる.

今回はWebpackを使ってHello Worldをコンソールに出力するモジュールをロードしてみる.

Webpackのインストール

npm install webpack -g

ソースコード

greeter.js
function Greeter() {
  this.greet = "Hello world";

  this.exec = function() {
    console.log(this.greet);
  };
}

module.exports = Greeter;

main.js

(function(){
  var G = require('./greeter.js');
  var greeter = new G();
  greeter.exec();
})();

Build

以下のコマンドでmain.jsにgreeter.jsを読み込ませたJSファイルを出力する.
webpack ./main.js bundle.js

以下のようにHTMLファイルからbuild成果物のbundle.jsを読み込ませる.

index.html

<html>
  <head>
    <meta charset="utf-8">
  </head>
  <body>
    <script type="text/javascript" src="bundle.js" charset="utf-8"></script>
  </body>
</html>

ブラウザで確認するとコンソールにHello Worldが出力される.
